# rag-book-code

**《산골 농부의 RAG》** — 책에 나오는 예제 코드 저장소입니다.

전남 담양의 산골에서, 밤마다 클로드와 함께 논문 수만 편을 읽는 RAG 시스템을 만들었습니다. 이 저장소는 그 여정을 담은 책의 **실습 코드**를 장별로 정리한 곳입니다.

> 이 책이 하고 싶은 말은 하나입니다.
> **"어려운 건 지식이 아니라, 시작할 용기였다."**

---

## 장별 코드

| 장 | 내용 | 폴더 |
|---|---|---|
| 1장 | 5분 만에 돌아가는 가장 단순한 RAG | [`ch01-simplest-rag/`](./ch01-simplest-rag) |
| 2장 | 죽으면 다시 살아나는 새 (불사조 파서) | `ch02-phoenix/` *(준비 중)* |
| 3장 | 이미 끝낸 것을 스스로 태우다 (NXML 재파싱) | `ch03-reparse/` *(준비 중)* |
| 4장 | 잘 되고 있는 걸 어떻게 아는가 (평가) | `ch04-eval/` *(준비 중)* |

각 폴더 안에 그 장의 실행 가능한 예제와 실행법이 들어 있습니다.

---

## 시작하기

**1장부터 시작하세요.** 그래픽카드 없는 평범한 노트북에서도 5분이면 돌아갑니다.

```bash
git clone https://github.com/JeaYoul/rag-book-code.git
cd rag-book-code/ch01-simplest-rag
```

필요한 것은 [Ollama](https://ollama.com)와 파이썬뿐입니다. 자세한 준비물은 책 1장에 정리해 두었습니다.

---

## 이 코드에 대하여

여기 있는 코드는 **가르치기 위한 최소 예제**입니다. 실제 운영 중인 시스템은 훨씬 복잡하지만, 복잡함은 배움을 가로막습니다. 그래서 각 장의 **핵심 원리만 남기고 덜어냈습니다.**

원리를 손에 넣으면, 나머지는 당신이 키우면 됩니다.
